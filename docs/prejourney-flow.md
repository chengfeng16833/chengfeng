## 流程明细

### 1. 游戏主界面 → 菜单(image1)
- 右上角菜单按钮(九宫格图标),**可能有红色感叹号、也可能没有,两种都要识别**。
- 锚:左侧竖排菜单文字 战斗/管理/总部/公会/商店/观测;右上资源条。
- 动作:点右上菜单按钮。

### 2. 菜单栏(image2)
- 弹出大面板,图标网格:付费商店/主线商店/作战/**旅程**/酒馆/培养广场…
- 动作:点"旅程"入口。

### 3. 难度选择 = 现有"选择旅程"INITIAL 画面(image3)
- 路线"星光引导者"。右下角三个难度按钮:**简单 / 一般 / 困难**,点击切换。
- 配置:`PreJourneyConfig.difficulty`(GUI/webui 可选)。
- 动作:点配置难度 → 点"开始"(现有逻辑已点开始)。

### 4. 角色选择 = 现有 CHARACTER_SELECT(image4~6)
- 右上角"排序"旁有**漏斗筛选按钮** → 弹职业筛选窗(image5)。
- 筛选弹窗职业(**UI 用语**):坦克/突击者/游侠/术师/刺客/辅助;底部 重置/确认。
  - ⚠️ 术语:docx 正文说"战士",UI 实际叫**突击者**(例:缇莉雅)。
- 用职业筛选解决**重名角色**问题(现有同名靠 ANOTHER/COSMIC 形态文字)。
- 配置:`PreJourneyConfig.profession`;之后按现有逻辑滚动找目标角色点"选择"。

### 5. 刻印选择 = 现有 BLESSING_SETUP(image7)
- 两个六边形"+"刻印槽,**位置不同,必须分两次进**(槽1一次、槽2一次)。
- 全部选好后右下"确认"**变蓝**才可点(锚:确认按钮颜色)。
- "自动装备"按钮存在但**不用**(用户早期规则)。

### 5.1 刻印操作 = 现有 BLESSING_CHOICE(image8~12)
1. 点**数值筛选**下拉 → 选"**能力值领域**"(卡片改为显示 力量:45 这类属性值)(image9)。
2. 点**属性筛选** → 弹窗按角色职业选属性(image10):
   - 辅助、大部分坦克 → **体力**;艾黛 → **韧性**;突击者(战士)/刺客/游侠/术师 → **力量**。
   - 与 `run_config.PreJourneyConfig.imprint_attribute()` 现有映射一致。
3. 筛选后网格**每排 5 个**(image11):按配置序号点第 N 个,默认第 1 个。
   - 序号→行列:第 4 个=第 1 排第 4 个;第 12 个=第 3 排第 2 个 → `row=(n-1)//5+1, col=(n-1)%5+1`。
   - 配置:`imprint_slot_1_index` / `imprint_slot_2_index`(两槽各一个)。
4. 点右下"确认"关闭回到刻印选择(image12)。

### 6. 支援卡选择 = 现有 JOURNEY_START(image13)
- 中间 5 张支援卡;**5 个圆点指示当前卡组(第几个亮=第几套)**,左右 `<` `>` 按钮切换。
- 需识别当前卡组点位 → 按 `support_deck` 配置切到目标卡组。
- 配置:`PreJourneyConfig.support_deck`(1~5)。

### 6.1 好友支援卡(新系统,image14~18)
- 入口:点卡列**最右卡位中心附近(不用识别,直接点)**(image14)→ 进支援卡选择界面。
- 列表里 OCR 找配置的**好友名字**(`friend_support_name`,如"B站老顾不烦")(image15~16)。
- 好友卡上有专属小图标(image17,6KB——**现成模板匹配素材**)。
- 点卡 → 支援卡详情界面(image18)→ 确认。

### 7. 开始训练(image19)
- 回到支援卡界面后点左下**"旅程起点"**(现有 JOURNEY_START 已点此按钮)→ 进入训练。
